**¡Bienvenidos a Minas Contigo!** Este es un proyecto abierto nacido de la idea de que el conocimiento complejo se entiende mejor cuando se construye de forma colectiva.

Aquí encontraras notas detalladas, ejercicios resueltos y conceptos clave de las asignaturas de matemáticas para ingeniería, todo escrito en **Markdown** y renderizado con **LaTex** para una calidad matemática impecable.

# 🚀 ¿Por qué este proyecto?

Las matemáticas en ingeniería pueden ser un reto. A menudo, los libros de texto son densos y las notas de clase están incompletas. Este proyecto busca ser el *"repositorio definitivo"* donde los conceptos se expliquen de forma clara, visual y precisa.

# 🤝 ¿Cómo puedes ayudar?

No necesitas ser un genio programando para contribuir. ¡Cualquier ayuda es valiosa!

1. ✍️ Agregar Contenido

> Si tienes apuntes claros sobre algún tema que falta o quieres complementar alguna de las notas ya existentes, puedes hacerlo directamente.

2. 🔍 Corregir Errores

> Si encuentras que alguien se equivocó en alguna formula, ecuación o incluso escribiendo una palabra, tu aporte puede ser vital para frenar que el conocimiento erroneo se pueda extender.

3. 🛠️ Mejorar el Formato

> Si conoces un poco de LaTex y podrias ayudar dandole formato y estetica a las formulas, tu ayuda también será ampliamente considerada.
